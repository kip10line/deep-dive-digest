## ğŸ› ï¸ Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/YOUR_USERNAME/deep-dive-digest.git
cd deep-dive-digest
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env.local` file in the root directory and add your API keys:
```env
# Google Gemini API
GEMINI_API_KEY=your_gemini_key

# YouTube Data API v3
YOUTUBE_API_KEY=your_youtube_key

# Google Custom Search (Web & News)
GOOGLE_PSE_API_KEY=your_google_pse_key
GOOGLE_PSE_CX=your_search_engine_id
```

### 4. Run Locally
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to start your research.
